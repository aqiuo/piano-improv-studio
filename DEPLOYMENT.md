## 📦 当前版本信息

**版本号：** v1.0.0  
**构建时间：** 2024年12月  
**主要功能：** 钢琴即兴演奏学习平台

## 🚀 部署步骤

### 1. 构建项目
```bash
npm run build
```

### 2. 部署文件
构建完成后，`dist` 目录包含所有部署文件：

- `index.html` - 主页面
- `assets/` - 静态资源（CSS、JS）
- `images/` - 图片资源
- `pdfjs/` - PDF.js 相关文件

### 3. 部署到服务器
将 `dist` 目录中的所有文件上传到您的Web服务器根目录。

### 4. 服务器配置
确保服务器支持：
- 静态文件服务
- HTML5 History API（用于Vue Router）
- HTTPS（推荐）
